# Chapter 17: Barrel Rate Analysis (2015-2025)

## Key Findings

- **Barrel rate stable at ~5%**: No significant change over decade
- **2015**: 5.15% | **2025**: 5.29%
- **Trend**: +0.004%/year (not significant, p=0.878)

## What is a Barrel?

A "barrel" is a batted ball with optimal exit velocity (98+ mph) and launch angle combination for maximum damage. Barrels have an expected batting average of .500+ and slugging of 1.500+.

*Batted balls analyzed: 2,198,495*
